# Screenshots

Drop desktop-app screenshots here. They are referenced by absolute
`raw.githubusercontent.com` URLs in `README.cli.md` (npm renders only absolute
image URLs, not relative paths), and may also be used in the root `README.md`.

Expected files (PNG, ~1200px wide, light or dark theme):

| File                 | Shows                                 |
| -------------------- | ------------------------------------- |
| `vault-overview.png` | Main window with the list of entries  |
| `entry-detail.png`   | A single entry / password detail view |

Add or rename as you like — just keep the filenames in sync with the `<img>`
tags in `README.cli.md`. Until these files exist on `main`, the images in the
published npm README will show as broken links.
